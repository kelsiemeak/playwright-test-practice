# playwright-test-practice
📝 Project Description

This is a beginner automation project using Playwright, created to build foundational skills in end-to-end testing.

The project focuses on learning how to:

* Navigate to web pages
* Interact with elements (clicking, typing)
* Write basic assertions to verify behaviour
* Use Playwright codegen and refine generated tests

The tests are intentionally simple and focus on clarity and consistency rather than complex framework design.

---

🎯 Purpose

The goal of this project is to:

* Transition from manual testing to automation
* Build confidence with Playwright
* Practice writing clean, readable tests
* Develop a habit of improving generated code

---

🚀 What’s included

* Basic Playwright test setup
* Example tests covering simple user flows
* Use of assertions to validate expected outcomes

---

📌 Notes

This is an early-stage learning project and will be improved over time as more advanced concepts (such as reusable functions, fixtures, and better test structure) are introduced.
